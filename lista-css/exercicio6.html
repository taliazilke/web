Como podemos incluir CSS em documentos HTML?


1. CSS Inline: O CSS inline é aplicado diretamente em um elemento HTML usando o atributo style. É uma forma simples e específica, mas de difícl manutençao e tende a se repetir deixando o código menos eficiente.

2. CSS Internal: O CSS internal é definido dentro de uma style tag na seção head do documento HTML. É eficiente para documentos pequenos por centralizar os estilos, mas é inadequado para váias páginas ou um grande documento.

3. CSS External: O CSS external é definido em um arquivo separado com a extensão .css, que é vinculado ao HTML usando a tag link.
